定时器创建常用方法

  • 方法一：performSelector

    2.0秒后自动调用self的hidHUD方法，并将参数传递给被调用的方法
    [self performSelector:@selector(hidHUD) withObject:@”参数” afterDelay:2.0];

  • 方法二：GCD

    2.0秒后自动调用block中的代码块
    dispatch_after(dispatch_time(DISPATCH_TIME_NOW, (int64_t)(2.0 * NSEC_PER_SEC)), dispatch_get_main_queue(), ^{
    // 2.0秒后自动调用此处block的代码块
    });

  • 方法三：NSTimer

    2.0秒后自动调用self的hidHUD方法，并传参数
    注意：这里的最后一个参数如果填写YES的话，会自动循环调用
    [NSTimer timerWithTimeInterval:2.0 target:self selector:@selector(hidHUD) userInfo:nil repeats:NO];

Plist（属性列表文件）

  • 一般可以使用属性列表文件(plist文件)存储NSArray或者NSDictionary之类的数据
  • 凡是参数名为file的传递的都是文件的全路径
  • 一个NSBundle对象对应一个资源包，用来访问与之对应的资源包内部的文件，可以获得文件的全路径

  • 加载plist数据
    利用mainBundle获得plist文件在主资源包中的全路径

    写法一：
    NSString filePath = [[NSBundle mainBundle] pathForResource:@”shops” ofType:@”plist”];
    写法二：
    NSString     filePath = [[NSBundle mainBundle] pathForResource:@”shops.plist” ofType:nil];
    // 加载Plist文件中的数据
    NSArray *array = [NSArray arrayWithContentsOfFile:filePath];

    • 注意：对于plist文件，如果处于黄颜色的文件夹下都不需要再填写路径的时候填写文件夹路径

一个控件不显示的可能原因

  • 没有设置上去frame
  • 没有添加到父控件上
  • 位置设置不对（超出屏幕）
  • hidden == YES
  • alpha <= 0.01
  • 没有设置背景色或内容
  • 内容颜色和父控件的背景色相同
