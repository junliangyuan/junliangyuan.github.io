全局常量的定义、const和指针的区别

标签（空格分隔）： IOS

[TOC]

常量的定义

建议使用以下方式定义常量

.h文件(声明常量)

.m文件(定义实现常量)

const声明和宏定义的区别

  • const声明的变量全局只有一份内存
  • 宏定义只是做一个字符的替换，每用到一次宏定义的变量，都会开辟一份临时的存储空间，相比const比较消耗内存

UIKIT_EXTERN 的用途

UIKIT_EXTERN本质上就是extern，可以扩展变量的作用范围。但是使用UIKIT_EXTERN系统会自动根据你的编译环境来自动识别extern。
注意：UIKIT_EXTERN只能在包含了<UIKIT/UIKIT.h>的框架下才能使用

const和指针的区别

以下三个语法的区别

  • (1) const int *p1;
  • (2) int const *p2;
  • (3) int *const p3;

由于指针*p的赋值方式有两种
第一种是：p = &a;
第二种是：*p = a;

(1)和(2)的效果是一样的。只能通过第一种方式来修改指针指向的变量

而(3)的方式是只能在一开始的时候指定一个变量，以后不能再指向其他变量。

其实主要是看const后面的变量是什么，只有const后面的变量无法修改
