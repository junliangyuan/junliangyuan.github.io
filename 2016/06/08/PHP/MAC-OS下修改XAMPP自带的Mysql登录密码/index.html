标签（空格分隔）： PHP

MAC下安装完XAMPP后，默认的PHPMyadmin是没有设置密码的，访问http://localhost/phpmyadmin/可以直接访问进去，我们通常希望给他设置一个密码。这个密码是作为数据库的连接密码。MAC OS系统下可如下设置

  • 1 启动Apache和Mysql服务
  • 2 打开终端，输入sudo /Applications/XAMPP/xamppfiles/xampp security

然后按照如图1-1所示输入对应的命令

注意 配置完以后必须重启Apache和Mysql服务。然后清空浏览器缓存。在浏览器中访问http://localhost/phpmyadmin/，这时候会有如图1-2提示没有访问权限的错误

解决办法是
